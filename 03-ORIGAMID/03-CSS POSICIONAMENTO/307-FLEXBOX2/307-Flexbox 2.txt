FLEXBOX 1
=========
-display: flex;
Os filhos passam a ter um tamanho flexível e ficam um ao lado do outro.

flex-wrap: wrap;
Caso não caiba todos os elementos em uma mesma linha, quebre para a próxima.

-gap
Define uma distância entre os elementos.

# PROPRIEDADE FLEX
==================
-flex-grow: 1;
Se esse elemento deve crescer para ocupar o espaço vazio. O 0 impede o crescimento, valores maiores que 0 funcionam como a unidade fr do css grid.

-flex-basis: 200px;
Valor inicial antes de distribuir o espaço em branco.

-O padrão flex-basis é automático, significa que o crescimento vai se dar no espaço que sobra (que esta vazio), ou seja, ele ja vai começar com uma base de acordo com o tamanho do conteúdo. Se eu reparar a distância distribuida no primeiro filme, ele cresceu em torno de 105px, e assim para os demais itens do elemento. Então ele esta distribuindo o espaço que sobra, APÓS O TAMANHO DO CONTEÚDO DO ELEMENTO. 

 AO MUDAR O FLEX-BASIS PARA ZERO (flex-basis:0;), todos os elementos agora tem 245px, fazendo com que o conteúdo de todos os elementos dos flex-items sejam iguais. Resumindo: ele desconsiderou o tamanho do conteúdo interno, porque tiramos o flex-basis:1; que é o padrão, e o colocamos em zero, passando a deixar todas as caixas no mesmo tamanho, independente das dimensões do conteúdo itnerno.

 AO MUDAR O FLEX BASIS PARA 200px, no minimo os elementos agora terão 200px de largura.
 Mas porque o elemento está com 240px de largura?
 -Porque nós temos um padding:20px no elemento h2, portanto ele soma o padding de 20px de um lado, 20px do outro + a largura de 200px que foi colocada no flex-basis, totalizando 240px. 

 OBS: RARAMENTE utilizaremos o flex-basis desta forma. Na maioria das vezes, ou deixa no auto, ou coloca no zero.

-flex-shrink: 0;
Caso exista um valor de base, o flex-shrink irá determinar se esse valor pode ser reduzido ou não. 0 significa que ele não pode ser reduzido.
Por padrão vem com valor de 1, o que significa que a palavra será  "quebrada" em linhas caso for diminuida.

-SHORTHAND
-flex: 1; 
Atalho para flex-grow: 1; flex-shrink: 1; e flex-basis: 0;

# ALIGN-ITEMS E JUSTIFY-CONTENT
===============================
O Align e Justify funcionam da mesma forma que no CSS Grid Layout. Lembre-se que essas propriedades só funcionam se existir espaço entre os elementos.
